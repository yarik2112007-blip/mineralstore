# MineralStore - интернет-магазин минералов

## Назначение
Веб-приложение для продажи минералов, кристаллов и натуральных камней.

## Стек технологий
- HTML5, CSS3, JavaScript (ES6+)
- Supabase (PostgreSQL + Auth)
- Git

## Установка и запуск
1. Клонировать репозиторий: `git clone https://github.com/yarik2112007-blip/mineralstore.git`
2. Открыть файл `index.html` в браузере
3. Для работы с базой данных нужен доступ к Supabase

## Основные функции
- Регистрация и вход с подтверждением email
- Каталог товаров с фильтрацией по категориям
- Поиск товаров по названию и описанию
- Корзина и оформление заказа
- Чат техподдержки
- Админ-панель (управление товарами, просмотр заказов)
- Отзывы на товары
